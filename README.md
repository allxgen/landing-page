## allxgen

Connecting all generations through experimental social products.

allxgen is a startup building social platforms that bridge generational divides. We create spaces where people of all ages can connect, collaborate, and create together.

### Getting Started

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the landing page.

### Contact

For inquiries and support, reach out to us at **admin@allxgen.com**

### Technology

This project is built with:
- [Next.js](https://nextjs.org) - React framework
- [Tailwind CSS](https://tailwindcss.com) - Utility-first CSS framework
- [TypeScript](https://www.typescriptlang.org) - Type-safe JavaScript
